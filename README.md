## Time Complexity (Big O notation)

![timecomplexity](/Users/luisgoate/Desktop/timecomplexity "Time Complexity")

| Operation          | Linked Lists          | Arrays   |
| :-----------------:|:---------------------:| :-------:|

| Printing data      | O(n)                  | O(n)     |
