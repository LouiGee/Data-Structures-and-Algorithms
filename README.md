## Linked Lists vs Arrays 

| Operation          | Linked Lists          | Arrays   |
| :-----------------:|:---------------------:| :-------:|
| Counting elements  | O(n)                  | O(1)     |
| Printing data      | O(n)                  | O(n)     |
