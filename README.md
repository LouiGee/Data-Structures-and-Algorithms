## Time Complexity (Big O notation)

| Algorithmic Runtimes (Fastest to slowest) |
| :----------------------------------------:|     
| O(1) Constant                             |
| O(logn) Logarithmic                       | 
| O(n) Linear                               |
| O(n^2) Polynomial                         |
| O(2^n) Exponential                        |
| O(N!) Factorial                           |

## Linked Lists vs Arrays 

| Operation          | Linked Lists          | Arrays   |
| :-----------------:|:---------------------:| :-------:|
| Counting elements  | O(n)                  | O(1)     |
| Printing data      | O(n)                  | O(n)     |
